# ✨ Coffee Shop Web Design — Portfolio Demo

Plantilla web pensada para negocio local. Diseñada para verse en todos los dispositivos.

## 👀 Páginas incluidas

- `index.html`: one‑page principal.
- `contact.html`: página de contacto con formulario demo.
- `menu.htlm`: página de menú.
- `privacy.html`: política de privacidad (texto breve de ejemplo).

## 🛠 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript vanilla (interacciones básicas)

## 📁 Estructura del proyecto

```
assets/
  images/
  icons/
css/
  styles.css
js/
  main.js
index.html
contact.html
privacy.html
```

## 🚀 Cómo verlo en local

Tienes dos opciones sencillas:

- Abrir directamente index.html en tu navegador

- Usar un servidor local (por ejemplo Live Server en VS Code)

## 🌍 Demo online (GitHub Pages)

Este proyecto se publica en GitHub Pages para compartir la demo.

## 🎨 Personalización rápida

- Cambiar los textos en `index.html`, `contact.html`, `privacy.html`, `menú.html`.
- Añadir imágenes en `assets/`.
- Ajuste de colores y tipografías en `css/styles.css` (variables en `:root`).

## ℹ️ Nota importante

Este proyecto es una demo/plantilla. El formulario no está conectado a un backend.
